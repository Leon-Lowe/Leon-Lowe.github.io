My experience making this project

The first major thing that I learned was how to separate my code into different files using header files and cpp files. This isn't something that needs to be done, but it definitely helps to maintain clean code. This means that I was able to create separate classes for the player, the enemies, and the props. Each of these then contained its own variables and functions. The most notable of these was the 'Tick()' function in the player and enemy class which held the code for everything they needed to do on each frame. For example, the player needed to get input, move, be drawn, and be animated every single frame.

The second major thing that I learned was using class inheritance. This meant that I could make a 'BaseCharacter' class which would contain any duplicate variables or functions that the player and enemy previously held. I also learned about virtual functions which would then allow me to override said functions in the child classes.

Other than those two major things, I learnt some more small things such as using pointers, better ways to render objects to the screen, and how to display numerical text to the screen.
