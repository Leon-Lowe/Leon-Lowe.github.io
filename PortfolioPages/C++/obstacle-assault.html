My experience making this project

This was my first real project made using Unreal Engine 5. As such, I made sure to keep it simple. For this game, all you need to do it make it to the other side of the map by using moving platforms.

In order to make these moving platforms, I created a script that I could attach to them called 'MovingPlatform' which would simply move the platform every frame along a given axis. I decided to focus a lot more on how to use the editor for this project.
