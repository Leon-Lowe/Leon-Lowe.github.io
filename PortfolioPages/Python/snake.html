My experience making this project

As this was my first project made in python, I had to figure out how to do a lot of things such as:

  • How do I create functions?
  • How do I create variables?
  • How do I create Classes?
  • How do I make a game window?
  • How do I make the game window stay open?
  • How do I get player input?

Thankfully, there are quite a lot of resources that could help me answer these questions and the pygame library helped with making a game window as well as getting input from the player.

First of all, I had a class called 'Game' which handled all of the main gameplay loop such as drawing, keeping the window open, and processing collisions. When the program is first opened, it will create a new 'Game' object, initialise it, and then call 'game.run'. The game class would then check to see if the game is currently running and, if it is, it would draw the current frame and collect any input from the player. Based on this input, it could then call 'Snake.move' which would move the snake. If the game class detects that the snake is colliding with the apple, it would tell the snake object to increase its length and tell the apple object to move to a new location.

For my first project in python, I think I did a pretty good job, especially by adding a pause function and a game over screen.
